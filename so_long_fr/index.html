MLX42 Library

MLX42 is a cross-platform graphics library written in C that uses OpenGL and GLFW for windowing and rendering. It provides a simple interface for creating windows, handling events, and rendering graphics.

Getting Started

Installation

cc program.c -I include -L lib -lmlx42 -lglfw -framework Cocoa -framework OpenGL -framework IOKit

For macOS, include the frameworks as shown. For Linux, link against required libraries: -lglfw -lm

Basic Window Creation


#include "MLX42/MLX42.h"
#include <stdlib.h>
#include <stdio.h>

int main(void)
{
    mlx_t* mlx = mlx_init(800, 600, "Hello world!", true);
    if (!mlx)
    {
        fprintf(stderr, "Error: mlx_init failed\n");
        return EXIT_FAILURE;
    }

    mlx_loop(mlx);
    mlx_terminate(mlx);
    return EXIT_SUCCESS;
}

Window Management

Window Control Functions

  • mlx_set_window_pos

    Sets the window position on screen

  • mlx_set_window_size

    Changes window dimensions

  • mlx_set_window_limit

    Sets minimum and maximum window size

Window Settings

  • MLX_STRETCH_IMAGE

    Enable/disable image stretching with window

  • MLX_FULLSCREEN

    Start in fullscreen mode

  • MLX_MAXIMIZED

    Start window maximized

Complete Example


#include "MLX42/MLX42.h"
#include <stdlib.h>
#include <stdio.h>

void my_key_hook(mlx_key_data_t keydata, void* param)
{
    mlx_t* mlx = param;
    if (keydata.key == MLX_KEY_ESCAPE && keydata.action == MLX_PRESS)
        mlx_close_window(mlx);
}

void my_resize_hook(int32_t width, int32_t height, void* param)
{
    printf("Window resized to: %dx%d\n", width, height);
}

int main(void)
{
    mlx_t* mlx = mlx_init(800, 600, "Example Window", true);
    if (!mlx) return EXIT_FAILURE;

    mlx_key_hook(mlx, my_key_hook, mlx);
    mlx_resize_hook(mlx, my_resize_hook, mlx);

    mlx_loop(mlx);
    mlx_terminate(mlx);
    return EXIT_SUCCESS;
}
